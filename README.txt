Les 2 fichiers proviennent d'un exercice contenu dans le cours "Apprenez à coder en JavaScript" d'OPENCLASSROOMS.
L'exercice permet d'utiliser un tableau et 2 boucles for. Le résultat s'affirera sur la console.
Le fichier HTML contient fait appel au fichier Javascript. Il contient son chemin et permet qu'il soit exécuté.

Attention : Pour utiliser les fichiers, il faut corriger le fichier .html et adapter le chemin contenu dans <script src="../js/mousquetaire.js"></script>
selon vos besoins
